## 7. Сводная карта реактивности

Когда ты меняешь поле на `/dashboard/teams/[id]` и жмёшь «Сохранить»:

| Источник изменения | Что триггерится | Задержка |
|---|---|---|
| `upsertTeam(team)` | `useTeams().teams` (context) → `teamTabs` memo → `teamColorFor` memo → `DayColumn` props → re-render | 0 мс (мгновенно) |
| `setCities([...])` | `useCities().cities` → `cityLookup` в `WeekView/DayColumn` → re-render + `CityPickerModal` | 0 мс |
| `upsertService(svc)` | `useServices().services` → `ServicePickerSheet` filter на следующее открытие | 0 мс (при следующем открытии sheet'а) |
| `setSchedules({...})` | `useSchedules().schedules` → `getTeamSchedule` memo → DayColumn workStart/workEnd | 0 мс |
| `setMasters(...)` | `useMasters().masters` → авто-лейбл + права | 0 мс |

Все контексты пишут в localStorage + обновляют React state. Cross-device sync нет (нужен Supabase).
